#My dotfiles

> These are my dotfiles, inspired by [Mathias dotfiles](https://github.com/mathiasbynens/dotfiles).

###Installation

```bash
git clone https://github.com/willianjusten/dotfiles.git && cd dotfiles

source bootstrap.sh

./install-main.sh
```

mongodb
```
mkdir -p /data/db
sudo chown -R `id -un` /data/db
```
